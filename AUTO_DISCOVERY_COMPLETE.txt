## Why `.local` Matters

```
IP Address: 192.168.1.100
  → Fixed number for this moment
  → Changes when WiFi restarts
  → URL breaks if IP changes

.local Hostname: raspberrypi.local
  → Resolves to current IP automatically
  → Always finds the Pi even if IP changes
  → Uses mDNS (works on all WiFi networks)
```

**In simple terms:**
- IP = Street address (changes)
- `.local` = Name of the house (stays same)

## Behind the Scenes

### How localStorage Works
```javascript
// First successful connection:
localStorage.setItem("apiBase", "http://raspberrypi.local:8000")

// Every page load:
const apiBase = localStorage.getItem("apiBase")
if (!apiBase) {
  show SetupPage  // First time only
}

// All API calls use this base URL:
axios.defaults.baseURL = apiBase
```

### Why It Persists
- Stored in browser, not on server
- Survives browser restarts
- Survives device restarts
- Survives WiFi changes
- Only cleared if user clears browser data
