🚀 QUICK START (3 steps)

1. Transfer to Raspberry Pi:
   scp -r ~/Desktop/Codes/ghahrApp pi@raspberrypi.local:/home/a/

2. Setup and configure:
   ssh pi@raspberrypi.local
   cd ~/ghahrApp/backend && python3 -m venv venv
   source venv/bin/activate && pip install -r requirements.txt
   nano .env  # Edit USER_A_PASSWORD, USER_Z_PASSWORD, ADMIN_PASSWORD

3. Build frontend and start:
   cd ~/ghahrApp/frontend && npm install && npm run build
   cd ~/ghahrApp && ./start.sh

Then open on ANY device:
   http://raspberrypi.local:8000

Why .local works:
  • mDNS resolution (multicast DNS)
  • Works even if WiFi restarts and IP changes
  • No DNS server needed
  • Works on Mac, Linux, Windows 10+, iOS, Android

📋 YOUR SETUP OPTIONS

Option 1: Quick Test (Development)
  • Run: ./start.sh
  • See logs in terminal
  • Stop: Ctrl+C
  • Restart: ./start.sh again

Option 2: Auto-Start (Production)
  • Install: sudo cp counter-app.service /etc/systemd/system/
  • Enable: sudo systemctl enable counter-app.service
  • Start: sudo systemctl start counter-app.service
  • View: sudo journalctl -u counter-app.service -f
  • Auto-restarts on fail and on boot

Option 3: Background (SSH Sessions)
  • Run: screen -S counter -d -m bash -c "cd ~/ghahrApp && ./start.sh"
  • View: screen -r counter
  • Detach: Ctrl+A, then D
